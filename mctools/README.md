# MCTools Help

Simple status cog for Minecraft bedrock servers.

# mcstatus
 - Usage: `[p]mcstatus`
 - Checks: server_only

Check the status of your Bedrock server.

# addserver
 - Usage: `[p]addserver <address> <port>`
 - Checks: server_only

Add an MC Bedrock server.

# delserver
 - Usage: `[p]delserver <name>`
 - Checks: server_only
